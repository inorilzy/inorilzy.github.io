```
/etc/nginx/nginx.conf
中指向
include /etc/nginx/conf.d/*.conf;
include /etc/nginx/sites-enabled/*;
```
此处说明可以将单个网站配置文件放在conf.d 或者sites-enabled中，但 /etc/nginx/ 文件夹中除了有sites-enabled 还有sites-available文件夹，通常做法是将配置文件放在sites-available中 ，然后通过软链接到sites-enabled中。
```
ln -s 源文件 目标文件 
举例：
ln -s /etc/nginx/sites-available/colors /etc/nginx/sites-enabled/colors 
```
